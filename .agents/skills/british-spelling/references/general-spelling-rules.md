# General English Spelling Rules

Common spelling rules for correct British English. Useful when writing code identifiers, comments, documentation, and user-facing text.

## i before e except after c

Place **i** before **e** except after **c**, when the sound is "ee".

Follows the rule:
- believe, achieve, retrieve, field, shield, yield, piece, brief, chief, relief, thief

After c:
- receive, ceiling, deceit, conceive, perceive, receipt

Exceptions (breaks the rule):
- weird, seize, neither, either, protein, caffeine, species

## Doubling consonants before suffixes

Double the final consonant before adding **-ed**, **-ing**, or **-er** when:

1. The word ends in a single consonant
2. Preceded by a single vowel
3. The stress falls on the final syllable (or the word is one syllable)

| Base     | -ed        | -ing        |
| -------- | ---------- | ----------- |
| stop     | stopped    | stopping    |
| run      | —          | running     |
| begin    | —          | beginning   |
| permit   | permitted  | permitting  |
| refer    | referred   | referring   |
| plan     | planned    | planning    |
| drop     | dropped    | dropping    |
| commit   | committed  | committing  |
| occur    | occurred   | occurring   |
| prefer   | preferred  | preferring  |

Do **not** double when stress is on the first syllable:

| Base    | -ed      | -ing      |
| ------- | -------- | --------- |
| open    | opened   | opening   |
| visit   | visited  | visiting  |
| listen  | listened | listening |
| offer   | offered  | offering  |

**British exception:** double the **l** regardless of stress (see british-spelling-reference.md):
- travel → travelled, travelling
- cancel → cancelled, cancelling
- model → modelled, modelling

## Dropping the silent e

Drop a final silent **e** before a suffix beginning with a vowel:

| Base     | Suffixed     |
| -------- | ------------ |
| have     | having       |
| make     | making       |
| save     | saving       |
| write    | writing      |
| create   | creating     |
| provide  | providing    |
| waste    | wasting      |
| race     | racing       |
| use      | usable       |
| desire   | desirable    |

Keep the **e** before a suffix beginning with a consonant:

| Base     | Suffixed      |
| -------- | ------------- |
| hope     | hopeful       |
| care     | careful       |
| manage   | management    |
| state    | statement     |
| achieve  | achievement   |
| use      | useful        |

Exceptions (keep e before a vowel):
- notice → noticeable
- change → changeable
- courage → courageous

## Changing y to i

When a word ends in a consonant + **y**, change **y** to **i** before adding a suffix (unless the suffix starts with **i**):

| Base     | Suffixed      |
| -------- | ------------- |
| carry    | carried       |
| happy    | happiness     |
| beauty   | beautiful     |
| easy     | easily        |
| busy     | business      |
| reply    | replied       |
| tidy     | tidied        |
| apply    | applied       |

Keep **y** when the suffix is **-ing** (to avoid double i):
- carry → carrying
- apply → applying
- copy → copying

Keep **y** when preceded by a vowel:
- play → played, playing
- enjoy → enjoyed, enjoying
- delay → delayed

## Adding -ly

Usually just add **-ly**:

| Base     | Adverb       |
| -------- | ------------ |
| quick    | quickly      |
| clear    | clearly      |
| slow     | slowly       |
| complete | completely   |

If the word ends in **-le**, replace **-le** with **-ly**:

| Base       | Adverb       |
| ---------- | ------------ |
| simple     | simply       |
| possible   | possibly     |
| terrible   | terribly     |
| comfortable| comfortably  |

If the word ends in **-y**, change to **-ily**:

| Base   | Adverb  |
| ------ | ------- |
| happy  | happily |
| easy   | easily  |
| angry  | angrily |

If the word ends in **-ic**, add **-ally**:

| Base      | Adverb        |
| --------- | ------------- |
| basic     | basically     |
| automatic | automatically |
| specific  | specifically  |
| tragic    | tragically    |

Exception: public → publicly

## Plurals

**Most nouns:** add **-s**

**Nouns ending in -s, -sh, -ch, -x, -z:** add **-es**
- bus → buses
- wish → wishes
- match → matches
- box → boxes

**Nouns ending in consonant + y:** change **y** to **-ies**
- category → categories
- dependency → dependencies
- library → libraries
- query → queries
- factory → factories

**Nouns ending in vowel + y:** add **-s**
- key → keys
- array → arrays
- day → days

**Nouns ending in -f or -fe:** often change to **-ves**
- half → halves
- shelf → shelves
- knife → knives
- life → lives

**Nouns ending in -o:** some add **-es**, some add **-s**
- hero → heroes
- potato → potatoes
- photo → photos
- video → videos
